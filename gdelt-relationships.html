⚡ horizon.gdelt.* gdelt.router.ts
  • listquery
    Paginated events by status (cursor)
    → gdelt_events (SELECT by status)
  • setStatusmutation
    Set new|flagged|skipped on an event
    → gdelt_events (UPDATE status)
  • countNewquery
    Count events with status='new'
    → gdelt_events (COUNT)
  • getEventquery
    Single event by global_event_id
    → gdelt_events (SELECT)
  • webcutquery
    Fetch + extract readable text from URL
    → no DB (fetchReadable())
⚡ intel.* intelRouter.ts
  • feedquery
    Paginated document feed
    → gdelt_documents (published_at not null)
  • searchDocumentsquery
    Full-text title search (ILIKE)
    → gdelt_documents (WHERE title ILIKE)
  • coveragequery
    Mention count + headlines for an event
    → event_mentions JOIN documents
    → gdelt_events (num_mentions, avg_tone)
  • eventsByDocUrlquery
    Events that mention a given document URL
    → event_mentions JOIN events JOIN documents
    WHERE num_mentions > 15
  • listEventsquery
    Paginated events + event code names
    → gdelt_events + event_codes (LEFT JOIN)
    optional actor/geo ILIKE search

End-to-end call flows
UI component / trigger tRPC procedure Tables read/written Notes
SubNavigationBar → horizon.gdelt.countNew → gdelt_events COUNT WHERE status='new', polled to show nav badge
TriageScreen · initial load → horizon.gdelt.list (×2) → gdelt_events Two concurrent infinite queries: status=new and status=flagged
FlaggedEventCard load → intel.eventsByDocUrl → event_mentions ⋈ events ⋈ documents Fired per flagged card; reveals related events for that source URL
TriageScreen · Flag / Skip → horizon.gdelt.setStatus → gdelt_events (UPDATE) Invalidates list + countNew; unflag/reflag also use setStatus
WebcutScreen · load → horizon.gdelt.getEvent → gdelt_events (SELECT 1) Renders event metadata header
WebcutScreen · article text → horizon.gdelt.webcut → — (fetchReadable only) Server-side fetch of source_url; no DB read
IntelEventsPage · scroll → intel.listEvents → gdelt_events ⋈ event_codes Cursor-paginated; optional ILIKE filter on actor/geo; num_mentions ≥ 1
IntelEventsPage · EventDetailModal → intel.coverage → event_mentions ⋈ documents + events Shows mention count, avg tone, and linked article headlines
Intel feed / search pages → intel.feed / searchDocuments → gdelt_documents Document-centric view; sortable by added or published date
Cron / Admin panel → POST /internal/jobs/gdelt
admin.runGdelt		 → events · event_mentions · documents (UPSERT) Advisory lock prevents concurrent runs; status never overwritten on upsert
